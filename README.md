Netflix Movies & TV Shows — Data Cleaning & EDA

Overview

This project involves data cleaning and exploratory data analysis (EDA) on the Netflix Movies & TV Shows dataset from Kaggle, completed as part of the AnalystLab Africa Data Analytics Internship Batch B (June–August 2026).

Dataset

Source: https://www.kaggle.com/datasets/shivamb/netflix-shows

Tools Used

	•	Python
	•	Pandas
	•	Matplotlib
	•	Seaborn
	•	Google Colab

What Was Done

	•	Handled missing values
	•	Removed duplicates
	•	Standardized text and date formats
	•	Performed EDA with 5 analyses and 5 visualizations
	•	Exported cleaned dataset

Key Insights

	1.	70% of Netflix content is Movies
	2.	Content peaked in 2019
	3.	USA is the top content producer
	4.	TV-MA is the most common rating
	5.	International Movies is the most popular genre
